How it works
Local-only mode
  • Everything is saved to this browser (localStorage).
  • No internet, no login, no database required.
  • Use Export CSV if you need to share or archive.
Logging flow
  1. Select Date, Hour block, and Packer/Station.
  2. Enter Part #, choose Pack type, then enter Qty.
  3. Optional: add Job Ref, Order Qty, and Notes.
  4. Click Save entry (or Save & stay to advance the hour block).
Shift schedules
The hour block dropdown is split into Day Shift (7am–3:30pm) and Night Shift (4pm–12:30am). The current block is auto-selected based on the time of day. Save & stay advances through all blocks in order across both shifts.
Daily totals
Totals uses adjusted hours (break blocks are weighted) to calculate pcs/adj-hr. Break multipliers: 15m break = 0.75 hr, 10m break = 0.833 hr.
Pack type filter
  • The filter (All / Stock / Special / Rework) affects: Today's table, Today totals, the weekly counter, weekly target planner, and the daily summary graph.
  • The weekly planner assumes a Mon–Sat workweek with a weekly target of 2,500.
  • Edit/Delete always applies to the underlying entry (even if you're filtered).
