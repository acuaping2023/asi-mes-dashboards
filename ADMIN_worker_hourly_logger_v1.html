How it works
v1.3 changes
  • Night shift hour blocks corrected: After the 9pm break, blocks are now 9:30pm–10:30pm, 10:30pm–11:30pm, and 11:30pm–12:30am (10m break, weight 50/60).
  • Standard value now required: std_per_hour_100 must be a positive number before saving. Saving a null standard created duplicate JOIN rows that doubled every worker's production count on that part.
Hour block weights (fraction of full hour worked)
  • 9am–10am (15m break) = 45/60
  • 1:30pm–2:30pm (10m break) = 50/60
  • 6:00pm–6:45pm (15m break) = 45/60
  • 11:30pm–12:30am (10m break) = 50/60
  • All other blocks = 1.0 (full hour)
  • Lunch breaks (12:00pm–12:30pm day shift, 9:00pm–9:30pm night shift) are not loggable blocks
