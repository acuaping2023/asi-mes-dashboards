How it works
This dashboard reads daily worker output and calculates efficiency against an engineering baseline (75% or 100%). The Shift filter reads shift assignments from the worker directory — set in the Logger's Manage modal. Filters update all views: summary cards, bar chart, trend line, leaderboard, and detail table. Selecting a worker enables the Worker Snapshot panel and 1:1 PDF export. Bar chart labels rotate automatically when there are many workers to prevent overlap. Demo mode loads safe sample data without connecting to the backend.
Tip: Use the Shift filter to compare day vs night shift performance independently.
