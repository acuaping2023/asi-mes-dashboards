How it works
This dashboard reads daily worker output and calculates efficiency against an engineering baseline (75% or 100%). The Shift filter reads assignments from the worker directory set in the Logger's Manage modal. Shift assignments are loaded once at startup and kept fresh via realtime — no race conditions. Bar chart labels rotate automatically with many workers. Demo mode loads safe sample data without connecting to the backend.
Tip: Use the Shift filter to compare day vs night performance independently. Click Refresh to force-reload shift assignments.
